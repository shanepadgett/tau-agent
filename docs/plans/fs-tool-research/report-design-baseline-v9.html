what the report proves

No vibe reads.

≤8k

Safe automatic read. Small enough that a full file read is cheap and unlikely to wreck context.

8–40k

Range read. The file may be useful, but the agent should take a slice before swallowing it whole.

40k+

Manual decision. Large files need intent, not optimistic context stuffing.

vendor/generated

Never auto. Generated mass is terrain to route around unless the user asks for it directly.
