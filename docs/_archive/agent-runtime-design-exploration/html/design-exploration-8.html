Threshold order book

≤8k Safe read candidate. Small enough for automatic inspection with ordinary context discipline.
8–40k Range read. Pull slices, symbol paths, or summaries before paying full context cost.
40k+ Manual. Treat as terrain, not text. Use targeted extraction or human decision.
never Vendor and generated files stay out of automatic read paths unless explicitly requested.
