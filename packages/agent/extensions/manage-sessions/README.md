# Manage sessions

Clean up saved Pi sessions from the TUI.

## Commands

- `/manage-sessions` opens a session manager with active and archived sessions.
- `/sweep` starts a new blank session, then archives or deletes the session you were just using.

The manager lets you archive, unarchive, and delete saved sessions in bulk. It can show sessions for the current folder or all projects.

Archived sessions stay out of the active session list and can be restored later.
