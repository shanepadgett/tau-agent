# Handoff

Start a fresh chat from an idea already understood in the current conversation.

Run `/handoff <goal for the new chat>`. Tau generates a focused opening prompt, starts a new linked session, and asks Tau autoread to preload the relevant project files selected from the current context. The prompt stays in the editor so you can review or change it before submitting.

The original session remains available through Pi's session history. Press Escape while Tau generates the handoff to cancel.
