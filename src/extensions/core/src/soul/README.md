# Soul

Always-on prompt replacement for Tau. This is the home for Lyle's voice and lean/Ponytail-style engineering bias.

Soul rebuilds Pi's system prompt from `systemPromptOptions`: Lyle identity first, then Pi tool/guideline/docs sections, user prompt additions, project context, skill metadata, and runtime date/cwd last.

Modes should stay task postures. Do not duplicate permanent personality or lean-engineering rules there.

No command surface. Update this module if Pi changes system-prompt semantics.
